# 🍽️ Perpex Internship Project

This project is a **Full-Stack Web Application** built as part of my internship at **Perpex**.  
It includes a **Django backend** (with JWT authentication) and a **Vite + React frontend**.

---

## 🚀 Features

- 🔑 **Authentication (JWT)**
- 🖥️ **Modern Frontend** using React + Vite
- 🎨 Clean UI with reusable components
- ⚡ API Integration with Django
- 🗂️ Modular project structure

---

## 🛠️ Tech Stack

**Frontend**  
- React (Vite)  
- Tailwind CSS / CSS Modules  

**Backend**  
- Django Rest Framework (DRF)  
- JWT Authentication  

**Others**  
- Git & GitHub for version control  
- Python (venv) for backend dependencies

## ⚙️ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/Rohit45-dot/Perpex_Internship.git
cd Perpex_Internship

cd restaurant
python -m venv env
source env/bin/activate   # (Linux/Mac)
env\Scripts\activate      # (Windows)

pip install -r requirements.txt
python manage.py migrate
python manage.py runserver

Backend will run on:
👉 http://127.0.0.1:8000/

Frontend (React + Vite)
cd frontend
npm install
npm run dev

Frontend will run on:
👉 http://localhost:5173/
